# NLP Course – ENSAE

This repository contains coursework and final project for the **Natural Language Processing (NLP) course at ENSAE Paris**.

## Structure

```text
Lab/        # Solutions to course labs
Project/    # Final NLP project
```

The **Project** analyzes French electoral manifestos using NLP methods to study whether political discourse is mainly structured by **party affiliation** or by **candidates' socio-professional characteristics**.
The analysis includes text preprocessing, topic modeling (LDA), and regression analysis.
